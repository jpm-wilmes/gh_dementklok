Dit programma (berichtwim.php) is een klok voor mensen met geheugenverlies. Het heeft de volgende functies:
- De tijd in het kwadrant linksboven (zo groot mogelijk)
- Dag, dagdeel en datum in het kwadrant rechtsboven
- drie tekstregels die uit een database komen in de onderste helft

Een tweede programma berichtwim.txt dat de data uit de database kan wijzigen. De database moet worden aangesproken via dbase.php

In de database staat 1 record met 3 velden plus een id (AI), elk voor 1 bericht met id=1

De klok draait op een tablet die aan een aantal eisen moet voldoen.
- opladen moet worden beperkt ter bescherming van de batterij
- het apparaat moet automatisch 's nachts kunnen herstarten
- het apparaat moet in kiosk modus draaien om interventies van gebruikers te voorkomen bijgaande link is een 
  goed werkende oplossing: https://play.google.com/store/apps/details?id=de.ozerov.fully
  let op dat er een standaard url wordt geopend waardoor de browser niet zichtbaar is
